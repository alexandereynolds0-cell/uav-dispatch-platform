## 一、系统概述

### 1.1 平台功能模块

本平台是一个无人机调度撮合平台，主要服务三类用户：

| 用户类型 | 说明 | 主要功能 |
|---------|------|----------|
| 客户 | 有作业需求的用户（农户、业主） | 发布任务、支付信息费、查看飞手联系方式、评价 |
| 飞手 | 提供无人机服务的个人或团队 | 接受任务、上传飞行数据、完成作业 |
| 管理员 | 平台运营人员 | 用户管理、任务审核、禁飞区配置、数据统计 |

### 1.2 核心业务流程

```
客户发布需求 → 平台计算信息费 → 客户支付 → 展示飞手联系方式 → 飞手接单 → 线下作业 → 客户评价
```

| 角色 | 权限说明 |

## 三、用户管理

### 3.1 查看用户列表

**路径**：用户管理 → 用户列表

功能说明：
- 支持按用户名、手机号、邮箱搜索
- 支持按角色筛选（客户/飞手/管理员）
- 支持按状态筛选（正常/禁用）
- 分页显示，每页50条

展示信息：
- 用户ID、昵称、手机号、邮箱
- 角色（客户/飞手）
- 注册时间、最后登录时间
- 状态（正常/禁用）

### 3.2 查看用户详情

**路径**：用户管理 → 用户列表 → 点击用户

展示信息：
- 基本信息（ID、昵称、头像、手机、邮箱）
- 角色信息
- 客户：企业名称、联系人、地址
- 飞手：实名认证状态、等级、评分、完成任务数
- 操作日志

### 3.3 禁用/启用用户

**路径**：用户管理 → 用户列表 → 操作 → 禁用/启用

注意事项：
- 禁用后该用户无法登录
- 飞手被禁用后将无法接单
- 禁用管理员账号需要超级管理员权限

### 3.4 修改用户角色

**路径**：用户管理 → 用户列表 → 操作 → 修改角色

可修改为：客户、飞手、管理员

### 3.5 删除用户

**路径**：用户管理 → 用户列表 → 操作 → 删除

注意事项：
- 删除为软删除，数据不会真正删除
- 删除后用户信息将被匿名化
- 需超级管理员权限

## 四、飞手资质管理

### 4.1 资质审核列表

**路径**：飞手管理 → 资质审核

展示信息：
- 飞手姓名、手机号
- 资质类型（无人机执照、运营证书等）
- 证书编号、有效期
- 提交时间、状态（待审核/已通过/已拒绝）

### 4.2 审核资质

**路径**：飞手管理 → 资质审核 → 点击审核

审核操作：
- **通过**：资质验证通过，飞手可正常接单
- **拒绝**：填写拒绝原因，飞手可重新提交

资质类型：
| 类型 | 说明 | 审核要点 |
|-----|------|----------|
| 无人机驾驶执照 | AOPA/UTC/ASFC等 | 证书编号、有效期 |
| 运营许可证 | 营业执照 | 经营范围、有效期 |
| 保险 | 无人机保险 | 保险公司、保额 |
| 设备登记 | 无人机注册号 | 设备型号、序列号 |

### 4.3 飞手实名认证

飞手实名认证状态：
- **未认证**：未提交身份证信息
- **认证中**：已提交，等待审核
- **已认证**：实名认证通过

## 五、任务管理

### 5.1 任务列表

**路径**：任务管理 → 任务列表

筛选条件：
- 状态：草稿、待审核、已发布、推送中、已接受、进行中、已完成、已取消、争议中
- 类型：喷洒、吊运
- 时间范围

展示信息：
- 任务ID、标题
- 任务类型
- 发布用户
- 地点、面积/重量
- 预算金额
- 状态、发布时间

### 5.2 任务详情

**路径**：任务管理 → 任务列表 → 点击任务

详细信息：
- 任务基本信息
- 发布用户信息
-  Assigned飞手信息（如果有）
- 推送历史（推送给哪些飞手）
- 订单信息
- 飞行数据（如果有）
- 评价信息（如果有）

### 5.3 任务操作

| 操作 | 说明 | 适用状态 |
|-----|------|----------|
| 审核通过 | 审核后发布任务 | 待审核 |
| 发布 | 手动发布任务 | 草稿、已审核 |
| 开始推送 | 将任务推送给飞手 | 已发布 |
| 强制取消 | 管理员强制取消任务 | 进行中、已接受 |

### 5.4 推送管理

任务推送给飞手后，可以查看：
- 推送时间
- 飞手响应状态（待响应、已接受、已拒绝、已过期）
- 响应时间

## 六、禁飞区管理

### 6.1 禁飞区列表

**路径**：系统配置 → 禁飞区管理

展示信息：
- 区域名称
- 类型（多边形/圆形）
- 风险等级（低/中/高/严重）
- 状态（启用/禁用）
- 创建时间

### 6.2 添加禁飞区

**路径**：系统配置 → 禁飞区管理 → 添加禁飞区

添加方式：
1. **地图绘制**：在地图上绘制多边形或圆形
2. **手动输入**：输入坐标点

参数说明：
| 参数 | 说明 |
|-----|------|
| 区域名称 | 禁飞区名称，如"南宁吴圩机场" |
| 类型 | 多边形或圆形 |
| 坐标 | 多边形输入顶点坐标，圆形输入中心点和半径 |
| 风险等级 | 影响颜色显示和提示文案 |
| 描述 | 详细说明 |
| 颜色 | 地图显示颜色 |
| 启用状态 | 是否生效 |

### 6.4 禁飞区设置

**路径**：系统配置 → 禁飞区设置

配置项：
- **允许在禁飞区发单**：是否允许用户在禁飞区内发布任务
- **显示警告**：在禁飞区内是否显示警告提示
- **警告标题**：警告弹窗标题
- **警告内容**：警告弹窗内容

### 6.5 用户发单检查

当用户在禁飞区内发单时：
- 如果"允许在禁飞区发单"为关闭：阻止发单，返回错误
- 如果"允许在禁飞区发单"为开启：允许发单，但显示警告

## 七、收费配置

### 7.1 定价参数

**路径**：系统配置 → 收费配置

可配置参数：

| 参数 | 默认值 | 说明 |
|-----|-------|------|
| 喷洒抽成比例 | 10% | 喷洒服务信息费抽成 |
| 喷洒最低费 | 5元 | 喷洒服务最低信息费 |
| 吊运抽成比例 | 8% | 吊运服务信息费抽成 |
| 吊运最低费 | 10元 | 吊运服务最低信息费 |
| 喷洒参考价 | 10元/亩 | 展示给用户的参考价 |
| 吊运参考价 | 2元/公斤 | 展示给用户的参考价 |

### 7.2 排序权重配置

**路径**：系统配置 → 排序配置

飞手排序因子权重（总和需等于100%）：

| 因子 | 默认值 | 说明 |
|-----|-------|------|
| 距离权重 | 30% | 距离越近越优先 |
| 评分权重 | 30% | 评分越高越优先 |
| 响应率权重 | 15% | 响应率越高越优先 |
| 活跃度权重 | 10% | 最近活跃时间越近越优先 |
| 完成率权重 | 15% | 任务完成率越高越优先 |

## 十三、常见问题

### Q1：如何添加新的禁飞区？
A：在禁飞区管理页面点击"添加禁飞区"，可以在地图上绘制多边形或圆形区域。

### Q2：为什么用户无法发单？
A：可能原因：1) 用户在禁飞区内且不允许发单；2) 用户账号被禁用；3) 账户余额不足。

### Q3：如何修改抽成比例？
A：在收费配置页面修改相应参数，修改后立即生效。

### Q4：飞手资质审核需要多长时间？
A：建议1-2个工作日内完成审核。

### Q5：如何查看聊天记录？
A：在消息记录页面按用户ID或任务ID搜索。
